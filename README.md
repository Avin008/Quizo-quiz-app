## Getting Started

- Clone the project

```bash
https://github.com/Avin008/Quizo-quiz-app.git
```

- Go to the project directory

```bash
  cd Quizo-quiz-app
```

- Install dependencies

```bash
  npm install / npm i
```

- Start the server

```bash
  npm start
```
